README - Reto Esteganografía (Nivel Principiante)
------------------------------------------------------------------

Introducción
---------------
¡Bienvenidos al reto de Esteganografía! Este desafío tiene como objetivo introducirte a las técnicas de ocultación de información en archivos multimedia por medio de los metadatos, específicamente en imágenes. En este reto, deberás descubrir una flag oculta en una imagen utilizando una herramienta de esteganografía.

Objetivo del reto:
- Encontrar la flag oculta en la imagen utilizando la herramienta Hex Editor Neo.

Herramientas a Utilizar:
- Hex Editor Neo: Herramienta para analizar imágenes en busca de esteganografía.

Instrucciones Paso a Paso
----------------------------

Paso 1: Abrir la imagen
- Abre la imagen.

Paso 2: Analizar la imagen para encontrar la flag
- Utiliza la herramienta Hex Editor Neo para buscar la flag oculta en la imagen.
  
Paso 3:Ingreso de la Flag
----------------------
- Una vez que hayas encontrado la flag, ingrésala en la flag correspondiente.
- Asegúrate de colocar la flag tal cual se muestra, incluyendo la estructura `flag{...}`.

Reglas del Reto
-------------------
1. NO USAR INTELIGENCIA ARTIFICIAL.
2. Solo se debe usar Hex Editor Neo para buscar la flag.
3. No se debe modificar el archivo de imagen proporcionado (`.png`).
4. No se debe de compartir la flag con los demás equipos.

Consejos Finales
-------------------
- Tómate tu tiempo y asegúrate de explorar a fondo todo lo proporcionado en esta introducción de tu equipo (interfaz).
