README - Reto Base64 (Nivel Principiante)
------------------------------------------------------------------

Introducción
--------------
¡Bienvenidos al reto de Base64! En este desafío, deberás decodificar el contenido de un archivo que está codificado en Base64 para revelar una flag oculta.

Objetivo del reto:
Decodificar el contenido en Base64 del archivo para revelar la flag oculta.

Herramientas a Utilizar:
https://www.base64decode.org/: Herramienta para decodificar texto codificado en Base64.

Instrucciones Paso a Paso
----------------------------

Paso 1: Abrir el archivo

- Abre el archivo publico.txt.

Paso 2: Copiar el contenido

- Copia todo el texto que contiene el archivo.

Paso 3: Visitar la herramienta de decodificación

- Visita https://www.base64decode.org/.

Paso 4: Aplicar la operación "Conjunto de Caracteres de Origen"

- Busca en todos los formatos la flag.

Paso 5: Obtener el resultado

- El resultado será un texto que contiene la flag.

Paso 6: Ingreso de la Flag

-Una vez que hayas encontrado la flag, ingrésala en la flag correspondiente.

-Asegúrate de colocar la flag tal cual se muestra, incluyendo la estructura flag{...}.

Reglas del Reto
-----------------

1. NO USAR INTELIGENCIA ARTIFICIAL.
2. Solo se debe usar https://www.base64decode.org/ para encontrar la flag.
3. No se debe modificar el archivo publico.txt.
4. No se debe compartir la flag con otros equipos.
